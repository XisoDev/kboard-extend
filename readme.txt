KBoard 기능 확장 플러그인입니다.

== Description ==

KBoard 워드프레스 게시판 플러그인의 기능을 확장하는 플러그인입니다.

주요 기능:
* 게시물 신고 기능
* 관리자 설정 페이지

== Installation ==

1. 플러그인 파일을 `/wp-content/plugins/` 디렉토리에 업로드하거나
2. 워드프레스 관리자 페이지에서 플러그인 > 새로추가 에서 'Kboard Extend'를 검색하여 설치하세요
3. 워드프레스 관리자 페이지에서 플러그인을 활성화하세요
